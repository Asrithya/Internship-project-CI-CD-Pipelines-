My Projects

Project 1

Airline Ticket Reservation System

The Airline Reservation System is a database management project aimed at automating the process of booking and managing airline tickets through a centralized and structured system. It enables passengers to search flights, book or cancel tickets, and view booking history, while providing administrators with tools to manage flight schedules, monitor seat availability, and update pricing. The system uses a relational database (such as MySQL or PostgreSQL) with well-defined entities like Flights, Passengers, and Bookings, applying normalization and constraints to ensure data integrity. By eliminating manual processes, the project enhances efficiency, accuracy, and user experience, and serves as a scalable foundation for real-world airline booking platforms.

Project 2

Text-Summarizer

The Text Summarizer is a Java-based application that automatically condenses large volumes of text into shorter, meaningful summaries using Natural Language Processing (NLP) techniques. It aims to help users quickly grasp the key points of lengthy documents by extracting and ranking the most significant sentences based on word frequency, relevance, and sentence structure. The system tokenizes the input text, filters out stop words, calculates sentence scores, and compiles a concise summary that retains the core message of the original content. Built using Java, and optionally enhanced with NLP libraries like OpenNLP or Stanford CoreNLP, this project showcases the practical application of Java in text processing and is especially useful in fields requiring quick content reviews, such as education, journalism, and legal analysis.

Project 3

Student Management System

The Student Management System is a web-based application designed to streamline and automate the management of student data within educational institutions. It provides a centralized platform where administrators, teachers, and students can efficiently access and manage academic information such as student registration, attendance, grades, course assignments, and personal profiles. Built using technologies like HTML, CSS, JavaScript, PHP, and a backend database like MySQL, the system enhances transparency, reduces paperwork, and ensures data accuracy. This project demonstrates the practical use of web development in creating scalable and user-friendly academic management solutions.
